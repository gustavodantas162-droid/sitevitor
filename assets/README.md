# Fotos reais e VSL do expert

Coloque aqui somente fotos reais do Vitor Guimaraes. Nao gere rosto, nao altere o rosto e nao use imagens de IA.

Arquivos esperados pela pagina:

- `vitor-hero.png` para o fundo do hero e poster da VSL.
- `vitor-editorial.png` para a foto principal da secao "Quem vai te ensinar".
- `vitor-closeup.png` para a imagem secundaria do expert.
- `vitor-human.png` para a secao "Por tras da aparencia".
- `vitor-authority.png` para a secao "Nao e para todo mundo".
- `manual-cover.png` para a capa/mockup do produto.
- `vitor-vsl.mp4` caso queira hospedar a VSL localmente.

Para ativar a VSL local, edite `PAGE_CONFIG.vslVideoUrl` no `index.html` para:

```js
vslVideoUrl: "./assets/vitor-vsl.mp4"
```

Para checkout externo, troque `PAGE_CONFIG.checkoutUrl` pelo link da Kiwify ou plataforma usada.
